# 算法探索——二维数组螺旋化输出

## 题目

为数组增加一个`snail_flatten`方法。

这个方法可以把如下`n*n`的数组：

```
// 输入
12 32 09 11 34
08 54 76 23 07
27 18 25 09 43
11 23 78 63 19
09 22 56 31 05
```

按如下的顺序变成一个一维数组：

```
1  2  3  4  5
16 17 18 19 6
15 24 25 20 7
14 23 22 21 8
13 12 11 10 9
```

```
// 输出
12, 32, 9, 11, 34, 7, 43, 19, 5, 31, 56, 22, 9, 11, 27, 8, 54, 76, 23, 9, 63, 78, 23, 18, 25
```

```ruby
# 例子

a = [
  [1, 2],
  [3, 4]
]
a.snail_flatten #=> [1, 2, 4, 3]

a = [
  [1, 2, 3],
  [4, 5, 6],
  [7, 8, 9]
]
a.snail_flatten #=> [1, 2, 3, 6, 9, 8, 7, 4, 5]

a = [
  [12, 32, 9, 11, 34],
  [8, 54, 76, 23, 7],
  [27, 18, 25, 9, 43],
  [11, 23, 78, 63, 19],
  [9, 22, 56, 31, 5]
]
a.snail_flatten #=> [12, 32, 9, 11, 34, 7, 43, 19, 5, 31, 56, 22, 9, 11, 27, 8, 54, 76, 23, 9, 63, 78, 23, 18, 25]
```

使用你喜欢的语言实现，要求有测试。
